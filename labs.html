Labs

Smaller-scale class assignments

  • Lab 1 - JavaScript and the DOM

    Create a web page that uses Javascript to interact with the DOM (document object model). This includes one CSS Property, one item that is not CSS, and should include two event listeners.

  • Lab 2 - HTML Forms

    Create an HTML form that uses HTML elements for the form, it's controls, and the submit button. Create two event listeners for the form in JavaScript.

  • Lab 3 - Javascript Debugging

    Find and fix the errors on a quiz and post the fixed version to your portal.

  • Lab 4 - APIs and JSON

    Create a webpage that uses the fetch API (built into the browser) to send requests any third party API, and display the JSON data that the third party API returns to you on the webpage in a creative fashion.
